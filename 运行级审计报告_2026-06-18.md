## 5. Vercel 是否使用 PostgreSQL

Vercel 环境变量（均为 Encrypted 状态，不可读明文值）：

```
DATABASE_URL       Encrypted    Production    30m ago
NEXTAUTH_SECRET    Encrypted    Production    30m ago
NEXTAUTH_URL       Encrypted    Production    30m ago
```

依据 `package.json` 第 7 行：

```json
"vercel-build": "cp prisma/schema.pg.prisma prisma/schema.prisma && npx prisma generate && npx next build"
```

- Vercel 构建时，`vercel-build` 脚本将 `schema.pg.prisma`（provider=postgresql）覆盖到 `schema.prisma`
- 编译出的 Prisma Client 为 PostgreSQL 引擎
- 运行时读取 `DATABASE_URL` 环境变量连接 Neon PostgreSQL
- Neon 项目 `cold-sun-68601240`（yunwu-brand-os）处于 active 状态，可写入

**结论：Vercel 生产环境使用 PostgreSQL（Neon）。**
